چجوری رباتمو به این لیست اضافه کنم ؟

    اول شما باید یه حساب گیت هاب داشته باشین و بتونین با گیت هاب و گیت کار کنین
    بعد شما یه ریسپانسیتوری توی گیت هاب درست میکنین و پروژتونو توش قرار میدین
    بعد از توی منوی code بخش https موس رو روی قسمت download ZIP قرار میدن راست کلیک میکنید و copy link address رو میزنین
    و بعدش وارد صفحه ی ریسپانسیتوری این سایت تو گیت هاب میشین و گزینه ی fork رو میزنین
    حالا یک کپی از این ریسپانسیتوری توی صفحه ی خودتون درست میشه
    الان باید این ریسپانسیتوری رو تو سیستم خودتون با استفاده ای گیت clone کنین
    بعد وارد فایل sources.json میشین از توی هر دسته ای که به ربات شما مربوط میشه بخش items رو پیدا میکنین و یک object با قالب زیر در اخر ارایه قرار میدین
    { "name": "نام ربات", "developer": "طراح", "URL": "ادرس فایل زیپ ربات در گیت هاب" }
    حالا یه merge pull request ایجاد میکنین و عنوانش رو افزودن ربات جدید قرار میدین
    request شما پس از تایید merge میشه و شما میتونین رباتتونو توی لیست ببینین ❤
